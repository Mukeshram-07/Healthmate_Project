## 🚀 Getting Started

### Manual Setup

```bash
# Clone the repo
git clone https://github.com/Mukeshram-07/Healthmate_Project.git
cd Healthmate_Project

# Create virtual environment
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run the application
python main.py
```
